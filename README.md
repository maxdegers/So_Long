## About
___

`so_long` is a small 2D game where players must navigate a map to collect all collectibles and reach the exit without being caught by enemies. The project combines basic game mechanics, window management, and event handling in C using the MiniLibX library, providing students with a fun way to practice their programming skills.

____
## Index
___
### Objectives

The objectives of `so_long` are to:
- Handle window management and graphic rendering.
- Work with textures and sprites.
- Implement basic gameplay elements such as player movement, collectibles, and obstacles.
- Design levels using a map file.
- Manage player input and game state.

### Features

- **Tile-based rendering**: The game world consists of a grid of tiles, each with its unique properties.
- **Collectibles and Obstacles**: Players must navigate through obstacles and collect items to complete the level.
- **Enemy AI**: Simple AI that challenges the player's ability to navigate the map.
- **Performance Optimization**: Efficient rendering and game loop to ensure a smooth gameplay experience.

## Usage

### Compilation

Compile the programs with:

```shell
make all
```
Running So_Long
Run the game, run:

```shell
./so_long map.ber
```
Replace map.ber with your map file.

## Controls

W, A, S, D - Move up, left, down, and right.

ESC - Exit the game.
